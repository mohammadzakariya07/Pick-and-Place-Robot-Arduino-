# Components List

## Main Components
- Arduino Uno / Nano
- 5 × Servo Motors (SG90 / MG996R)
- 5 × Potentiometers (10kΩ)
- Robotic Arm Frame (Base, Shoulder, Elbow, Wrist, Gripper)

## Power & Wiring
- External 5–6V Power Supply (for servos)
- Jumper Wires
- Breadboard or PCB

## Tools
- USB Cable
- Arduino IDE
